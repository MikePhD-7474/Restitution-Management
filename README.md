# Church Management System
## Restitution Church - Chisamba

A simple, offline-capable church management system built for rural communities with intermittent internet connectivity.

---

## 🚀 How to Run This Project

### Step 1: Copy Files to Your Project Folder
1. Copy all the files you downloaded into your `church-management` folder
2. You should have these files:
   - index.html
   - style.css
   - app.js
   - manifest.json
   - README.md

### Step 2: Open in VS Code
1. Open Visual Studio Code
2. Click File → Open Folder
3. Select your `church-management` folder

### Step 3: Start a Local Server
You need a local server to run this properly. Here's how:

**Option A: Using VS Code Extension (Easiest)**
1. In VS Code, click on the Extensions icon (left sidebar)
2. Search for "Live Server"
3. Install the "Live Server" extension by Ritwick Dey
4. Right-click on `index.html` and select "Open with Live Server"
5. Your browser will open automatically!

**Option B: Using Node.js**
1. Open Terminal in VS Code (View → Terminal)
2. Run: `npx serve`
3. Open your browser and go to: http://localhost:3000

## 📱 Features Currently Working

✅ **Member Registration** - Add new church members with full details
✅ **Offline Storage** - All data saved locally, works without internet
✅ **Member Directory** - View all registered members
✅ **Search** - Find members by name or phone number
✅ **Statistics** - See member count, gender distribution, baptism stats
✅ **Mobile Friendly** - Works on phones and tablets
✅ **Delete Members** - Remove members from the database

---

## 💾 How Data is Stored

- All data is saved in your browser's **localStorage**
- This means data stays on the device where you enter it
- Works completely OFFLINE - no internet needed
- Data persists even after closing the browser

**Important**: Data is stored locally on each device. If you use different devices, they will have separate databases (for now).

## 📱 Installing on Android Phone

1. Open the website in Chrome on your Android phone
2. Chrome will prompt "Add to Home Screen" or you can:
   - Tap the three dots menu (⋮)
   - Select "Add to Home Screen"
   - Name it "Church MS"
3. An icon will appear on your home screen like a regular app!
4. Works offline after first visit

## 🆘 Troubleshooting

**Problem**: Page looks broken or doesn't work
- Make sure all 4 files are in the same folder
- Try refreshing the page (Ctrl + F5)

**Problem**: Can't open the page
- Make sure you're using a local server (Live Server extension)
- Don't just double-click the HTML file

**Problem**: Data disappeared
- Check if you're using the same browser
- localStorage is browser-specific
